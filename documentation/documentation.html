Step 1 - Install Upic

In this part we will cover the installation of Upic on your own website hosting, from the files to the database.

- You need to have a web hosting already unless you want to install Upic locally. If you don't have a web hosting yet, I recommend Bluehost. You can start by the cheapest plan which is still awesome, for example, it's one of the only hosting which gives you SSD storage (it makes your website a lot faster and is even better for the upload features of Upic!). Here is how to do if you want to use Bluehost :

  1. Go to BlueHost by clicking here
  2. In their menu, click Hosting > Shared Hosting
  3. On this page, you'll have multiple plans. You can definitely start with the Basic one which will be enough for Upic
  4. Click on Select and enter your website domain name (for example Upic.com)
  5. Fill the form and in Package Information you can change the Account Plan to "One Year" if you want to pay less
  6. Once you have filled this form and got an access to your hosting panel, you can follow the rest of the documentation :-)


A. Transferring Files

With your web hosting infos, you should have some FTP login details. Use a software like FileZilla to connect to your FTP server and transfer the content of the folder named "Upic Files" to your server.

For allowing photos to be uploaded, you will also need to change the chmod of the folders uploads to 775.


B. Set Up the Database

On your web hosting panel, you should have an option to create a database and manage it through PhpMyAdmin.

Log-in to PhpMyAdmin and select your database, then go to the "Import" tab and pick the file named "database.sql" which is present in the folder named "Upic Database". This will create the required tables to use with Upic.


C. Update the config file

In the folder "Upic Files", go to "includes" and open the file "config.php". In this file you should update the lines related to your website and database config as follow. 

// MAIN URL
define("URL", "https://upic.yourdomain.com");
Change this line to your website URL 

// DB Host
define("DB_HOST", "localhost");

// DB Username
define("DB_USERNAME", "");

// DB Password
define("DB_PASSWORD", "");

// DB Name
define("DB_NAME", "");
Change these lines to your database config


Congrats! Your website is now ready to use!

Step 2 - Creating the first account & admin area

If you followed the documentation until now, your installation should work perfectly. To test if everything is fine, go to your website URL and go to the sign up page to register the very first account of your website.

The first account you create will automatically be granted with admin privileges. That's why, once logged in, you should see the red button called "Admin Area".

In the admin area, you will see different menu items, don't hesitate to play with each sections to better discover the script, even if they are pretty self-explanatory.

Step 4 - Customizing Upic

Here are a few details on the files and how to configure some parts of Upic.

- The main design code / CSS is present in the file css/custom.css. That's where you can change colors and most aspects of the design
- The top menu can be changed in the file templates/headers/global_header.php a templates/headers/admin_header.php for the admin side.
